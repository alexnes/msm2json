msm2json
========
Набор скриптов для извлечения данных из базы данных msm с сервера с настроенным MSM-Activate.

saved/B1 - пример выгруженных данных.  
GETGL.R - рутина на языке M; принимает имя глобала в виде параметра и отправляет в stdout все записи глобала в виде JSON-совместимых строк.  
MSMCmd.pl - скрипт, выполняющий M-команду на удаленном сервере.  
msm_json.py - python-модуль, предоставляет функции для чтения выгруженных данных и преобразование их в структуры языка.  

```./MSMCmd.pl "D ^GETGL(\"B1\")" >saved/B1```
